AWS Cognito + SAML IDP
SAML Authentication

The Flask application allows to initiate the login flow by either calling the /login endpoint or by calling any protected path in the application. AWS Cognito handles the User's login and is connected to the corporate Azure AD. It is important to note that the application will implement the authorization flow (as described in authorization-endpoint) and will then use the retrieved access_token to call the backend service.
