ECS + Flask Webapp
Graphical UI

The Flask application presents a graphical user interface and allows to initiate the login flow. It runs as a Docker container in an ECS Fargate cluster. It uses Flask under the hood, for its simplicity and ease of use. Any containerizable application framework would do. The ECS cluster is deployed from the Terraform module ecs_cluster. It is important to note that the application is deployed using a Blue / Green deployment (more details below).
